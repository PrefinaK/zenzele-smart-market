# Zenzele Smart Market 🚀
### *Be Self-Reliant. Build Your Future.*

An open-source, web-based entrepreneur marketplace built on the **Cardano blockchain**.

---

## 🌍 What Is Zenzele?

Zenzele (meaning "be self-reliant" in isiZulu) is a platform for unemployed people, small business owners, freelancers, creators, students, and everyday hustlers from **any country** to:

- 📣 Promote their skills, products, and services
- 🎙️ Record an audio pitch with background music
- 🏆 Mint an Entrepreneur NFT on Cardano
- 💰 Receive donations via Cardano
- 📚 Request training and mentorship
- 🔗 Share their profile on WhatsApp, TikTok, Facebook, Instagram, Telegram & X
- 🌐 Use the platform in 6+ languages

---

## 🚀 Quick Start

1. Clone or download this repo
2. Open `index.html` in your browser (no server needed for frontend demo)
3. For full backend, set up PHP/MySQL (see `/php/` folder)
4. Visit `pages/register.html` to create a profile
5. Or use the **Demo Login** on `pages/login.html`
